Fuse Bridge

Relay your Fuse guild chat to Discord —
automatically, silently, in the background.

Download Fuse Bridge

Windows  ·  No installer required  ·  Updates automatically

Getting Started

  1. 1 Download and run FuseBridge.exe. It sits in your system tray — no window needed. Right click -> Open when you need to access more info (Inventories, Map, Timers, etc.)
  2. 2 Right click icon in tray, Open -> Set EQ Directory and Link your Discord account (run a discord /linkclient command with a code from the app)
  3. 3 That's it. Guild chat and alerts appear in Discord. Fuse Bridge keeps itself up to date.
