Fuse Bridge

Relay your Fuse guild chat to Discord —
automatically, silently, in the background.

Download Fuse Bridge

Windows  ·  No installer required  ·  Updates automatically

Getting Started

  1. 1 Download and run FuseBridge.exe. It sits in your system tray — no window needed.
  2. 2 Right click icon in tray, Settings > General > Set EQ Install Directory
  3. 3 That's it. Guild chat and alerts appear in Discord. Fuse Bridge keeps itself up to date.

Features

  • Guild-stream redundancy Provides a backup stream of guild chat in case our bot goes down
  • FTE Messages See in guild-stream who won a mob quickly, depends on someone in zone running this client!
  • Character tracker Shows you where you left your characters, quick at-a-glance park information
